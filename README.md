# The Lonsdaleite Project

Harder than diamond.

A monolithic Linux kernel based on 6.18.20 LTS. (Will Change In the Future.)

- Faster than vanilla stock Linux
- More secure than linux-hardened
- Comes with its own extremely lightweight kernel-level antivirus (because user-space bloat is for bitches)
- GPLv2 licensed

When asked why it's called Lonsdaleite: "Because it's more durable than diamond."

Work in progress. Built by Michael.
